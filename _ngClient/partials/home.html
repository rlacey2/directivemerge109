This example shows a directive being EXTENDED twice to generate new directives, inheriting some but not all attributes of the original directive.
From the html on the directive useage set parameters.

Another feature is that the $scope variables are created via the html rather than in the controller, showing the two-way relationship. In this example $scope.person is geneated and shared between all the directives.
